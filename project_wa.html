Opdracht:

Er moet een user interface ontworpen worden, samen met een deel van de flow voor de chatbot die wijzer.amsterdam gebruikt. Met behulp van dit interface kunnen digibeten van 65+ hulp zoeken op de website.

Geleerd:

  • Figma
  • Notion

Beschrijving:

Dit project is gemaakt is mijn tweede jaar. Tijdens dit project zaten we in de corona pandemie. Voor mij was het belangrijk om ouderen te helpen om leren om te gaan met digitale middelen, zodat zij hulp konden zoeken bij onderwerpen die tijdens de pandemie van belang waren. Denk hierbij aan hoe videobellen werkt of wat zij kunnen doen wanneer zij zich eenzaam voelen.

Omdat digibeten niet bekend zijn met computers en niet weten hoe zij met digitale media om moeten gaan, heb ik in mijn user interface gebruik gemaakt van onboarding. Onboarding leert gebruikers hoe zij apps moeten gebruiken. In mijn design heb ik tooltips gebruikt, waarbij de gebruiker op een contextuele manier door de chatbot wordt begeleid.

Functies

  1. Knop voor chatbot Ezra:

    Wanneer de gebruiker voor het eerst op de website komt, laat de website weten dat de gebruiker gebruik kan maken van Chatbot Ezra. De notificatie legt de gebruiker uit wat de gebruiker daarvoor moet doen.

  2. Voorstellen chatbot Ezra:

    Chatbot Ezra stelt zich netjes voor en legt uit wat de chatbot doet.

  3. Uitleg rondleiding

    Wanneer de gebruiker ervoor kiest om de rondleiding te volgen, wordt de tooltip uitgelegd.

  4. Rondleiding

    De gebruiker kan de rondleiding op elk moment volgen. Ook kan de gebruiker ervoor kiezen om de rondleiding te pauzeren, stoppen of opnieuw te volgen.
